1. В .config вставляем строку подключения для бд
2. По умолчанию создаём базу REGISTRATION данных с Table_1  и полями
id (int) login nvarchar(50) password(bigint) history nvarchar(500)
3. Регистрируемся или входим
4. Для изменения ника или порта зайти в settings изменить данные и нажать кнопку Acept
после чего перезапустить приложение
5. Для написания собщения впишите его в нижнее поле и нажмите кнопку send
6. Если после выхода вы войдёте под свои логином и паролем вам 
будет доступна история переписки
PS Приложение используется для связи по ЛОКАЛЬНОЙ сети
